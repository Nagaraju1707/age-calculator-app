#  Smart Age Calculator
#  Author

Chilla Nagaraju  
Aspiring Data Analyst | Python Developer

A simple and elegant Streamlit web app that calculates your exact age in:

- Years
- Months
- Days
- Hours
- Minutes
- Seconds

##  Features

1. Date of Birth selection  
2. Exact age calculation  
3. Total hours, minutes & seconds  
4. Clean UI  

##  Built With

- Python
- Streamlit
- dateutil

##  How to Run Locally

1. Clone the repository:
   git clone https://github.com/nagaraju1707/age-calculator-app.git

2. Navigate to project folder:
   cd age-calculator-app

3. Install dependencies:
   pip install -r requirements.txt

4. Run the app:
   streamlit run app.py
